Printing Basics

The print() function is used to output data to the standard output device (your screen).

print("Hello, World!")
print(5 + 5)

Variables

Variables act as containers to store data values. In Python, a variable is created the moment you first assign a value to it.

name = "Alice"
age = 25
is_student = True

Math Operations

Python supports standard mathematical operations like addition, subtraction, multiplication, and division.

addition = 5 + 3        # 8
subtraction = 10 - 2    # 8
multiplication = 4 * 2  # 8
division = 16 / 2       # 8.0
exponentiation = 3 ** 2 # 9

Lists

Lists are used to store multiple items in a single variable. They are created using square brackets.

fruits = ["apple", "banana", "cherry"]
print(fruits[0]) # Output: apple

Loops

Loops are used to repeat a block of code. Python has for loops and while loops.

# For loop
for i in range(3):
    print(i) # Outputs 0, 1, 2

# While loop
x = 0
while x < 3:
    print(x)
    x += 1
